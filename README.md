# Shooter-Roguelike
Top-down shooter roguelike donde el jugador avanza por pisos de un dungeon generado proceduralmente. En cada piso hay NPCs (mercaderes, prisioneros, aliados dudosos) con los que puedes conversar en lenguaje natural. Los NPCs recuerdan lo que hiciste en la run, reaccionan a tus decisiones y pueden darte misiones emergentes.
